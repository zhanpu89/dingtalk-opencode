## 1. 测试范围

- **被测模块**：会话与上下文持久化（SessionStore + ContextStore）
- **被测接口**：6 个（SessionStore.get/set/flush/size + ContextStore.get/set/delete/clearAll）
- **用例总数**：23 条（单元测试 17 + 集成测试 6 + 安全测试 0 + 性能测试 0）
- **关联业务规则**：STO-REG-01、STO-REG-02、STO-REG-03、STO-REG-04、STO-REG-05、STO-REG-06
- **关联代码评审报告**：无
- **AC 追溯**：
  - AC-会话管理-01（首次自动创建会话映射）→ TC-STO-UNIT-001, TC-STO-INTG-001
  - AC-会话管理-02（复用已有会话映射）→ TC-STO-UNIT-002, TC-STO-INTG-001

---

## 2. 全局前置条件

1. 测试环境使用临时目录存放 JSON 文件，测试前后清理
2. 文件系统操作使用 Mock（fs.promises）隔离
3. 时钟使用 jest.useFakeTimers 控制防抖定时器
