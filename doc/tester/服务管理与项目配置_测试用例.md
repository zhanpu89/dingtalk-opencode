## 1. 测试范围

- **被测模块**：服务管理与项目配置（ServerManager + ProjectRegistry + Config）
- **被测接口**：9 个（ServerManager.start/stop/startProject/checkProject/list/get/disposeProject + ProjectRegistry.load/findById/findByName + Config.load）
- **用例总数**：24 条（单元测试 19 + 集成测试 5 + 安全测试 0 + 性能测试 0）
- **关联业务规则**：SVR-REG-01 ~ SVR-REG-08
- **关联代码评审报告**：无
- **AC 追溯**：
  - AC-项目服务-01（自动启动未启动项目）→ TC-SVR-UNIT-006, TC-SVR-INTG-001
  - AC-项目服务-02（复用已启动项目服务）→ TC-SVR-UNIT-007
  - AC-项目服务-03（异常退出后恢复）→ TC-SVR-UNIT-009

---

## 2. 全局前置条件

1. 所有子进程操作使用 Mock（child_process.spawn）隔离
2. 文件系统操作使用 Mock（fs）隔离
3. 健康检查 HTTP 调用使用 Mock（fetch）隔离
4. 端口检测使用 Mock（net.createServer）隔离
